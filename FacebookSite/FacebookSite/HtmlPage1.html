Fast ASP.NET Websites

Fast ASP.NET Websites is a practical, hands-on guidance to create faster, more efficient ASP.NET sites and applications. This book offers step by step .NET-specific examples showing you how to apply classic page optimization tips, ASP.NET-specific techniques, and ways to leverage new HTML5 features.

For more information about this book and other Manning titles, please visit the Manning website.

 

 

All code samples and snippets are provided with full support for Visual Studio 2012. This book leverages the power of ASP.NET 4.5.
